# claude-predefined-prompt

这是一个浏览器扩展插件，在 Claude 的输入框里面使用预定义的 Prompt。例如，输入`/commit`加空格，就会自动替换为`请帮我写一条Git commit message，内容是：`。

[点此安装](https://greasyfork.org/zh-CN/scripts/489838)
